This update includes some small improvements and bug fixes:

- Login can now share information with your main web browser. In other words, Ryuusei will try to log into the same account you're logged in with in your web browser, if any. This should hopefully reduce issues related to email confirmation links and Captcha failures. If you don't want to use the same login as in your web browser, long-press the login button in Ryuusei and press "Fresh Login".
- Fixed an issue with some error messages appearing as banners that should only be popup dialogs or full-screen
- Under the hood modernization of some screens
