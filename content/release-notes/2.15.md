This version brings some highly requested new features, plus other improvements:

- Advanced list entry editor! View and edit detailed info about your list items: start / end dates, rewatch count, advanced scores, and notes. You can find the advanced editor by going to an anime/manga detail screen, tapping the More / 3-dot menu button, then tapping Edit Entry. Or you can long-press items on the Dashboard and List screens.
- The Profile tab now shows your configured app accent color rather than your AniList profile accent color. Other users’ profiles continue to show their profile accent color.
- Profile “About” sections have better formatting for lists and quotes, and now support text selection and long-pressing links.
- Searching for media with an apostrophe in the name should more reliably find results
- If you have push notifications enabled, Ryuusei now notifies you if get logged out (because push notifications will not work until you log back in!)
- You can now view when your login token will expire under Settings > Account
- Fixed a crash on staff detail screens that could occur if navigating back while the screen was still loading
- Fixed missing role info on character list screens if a character has multiple voice actors
- Fixed an issue where markup tags were not handled correctly if they were not all lowercase (in other words, you may have seen things like Img(...) or Webm(...) incorrectly showing up in blocks of text)
