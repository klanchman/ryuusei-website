This update includes minor improvements and bug fixes:

- When filtering the List screen or editing an item, your custom lists are now sorted the same as on AniList.co, including any manual sorting you've set up
- On iOS 26, the Search page's input would change its height based on whether the paste button was being shown. This was annoying. The input now stays at the taller height all the time.
- Remember a couple updates ago when I "[possibly] fixed a [possible] crash on iOS 26"? The good news is that fix seemed to work. The bad news is it uncovered a couple of similar crashes in other parts of the app. This update should fix those!
